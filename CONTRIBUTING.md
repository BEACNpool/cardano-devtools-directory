# Contributing

Thanks for helping keep this directory useful.

## What this repo is
A curated, **developer-focused** directory of tools, libraries, and infrastructure for building on Cardano.

- Not a list of dApps/projects/tokens.
- Not a marketing directory.
- Goal: high-signal, easy to scan, and hard to let rot.

## How to add or update an entry
Open a PR that edits the relevant section(s) in `README.md`.

Include:
- **Name**
- **Repo URL** (preferred) or official docs URL
- **Language/stack** (if relevant)
- **One-line description** (what it does, for builders)

Strongly preferred:
- License
- Status: `active | maintenance | experimental | deprecated | archived`

## Inclusion guidelines (keep it fair)
We want this to feel inclusive while staying high-quality:

- Must be relevant to **developers building on/with Cardano**
- Must have a public home (repo or docs) that others can access
- Must be reasonably maintained *or* clearly labeled as deprecated/archived
- Keep descriptions factual; avoid hype language

## Style
- Alphabetize entries within a table where it makes sense
- Keep descriptions to ~1 sentence
- Prefer canonical upstream repos over forks (you can mention notable forks if relevant)

## What happens to low-quality entries
If something is spammy, dead, misleading, or purely promotional, it will be declined or labeled accordingly.

PRs that improve accuracy (links, descriptions, categories) are especially welcome.
