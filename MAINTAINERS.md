# Maintainers

This repo is hosted under **BEACNpool**, but it’s meant to be a community resource.

## Maintainer goals
- Keep the directory high-signal and up to date
- Be fair and consistent with inclusion criteria
- Prefer objective, verifiable facts over opinions/marketing

## Decisions
Maintainers may:
- Request changes to wording/tone
- Re-categorize entries
- Label entries as deprecated/archived
- Decline PRs that are spammy, misleading, or out of scope

## How to become a maintainer
Open an issue describing how you’d like to help (triage, link checks, category stewardship). Consistent quality contributions earn trust.
